== Description ==

Generator AI Now is a high-performance content engine for WordPress. It allows you to generate high-quality articles for any niche (Recipes, News, Reviews) using OpenAI (ChatGPT) and Google Gemini.

Key Features:
* Multi-Niche Support (Recipes, News, Reviews).
* Automatic CPT Discovery: Target any Custom Post Type on your site.
* Dynamic Category Selection via AJAX.
* Professional Image Generation with Niche-aware prompts.
* Error Logging System for API monitoring.
* Full SEO compatibility.

== Installation ==

1. Upload the `generator-ai-now` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'AI Generator' and add your API Keys.
